# termin-components-collision

Collision component package for connecting entity components to collision primitives/worlds.

Связанные документы:

- [termin-components](../../docs/index.md)
- [termin-collision](../../../termin-collision/docs/index.md)
- [termin-scene](../../../termin-scene/docs/index.md)

## Основные области

- Component headers in `components/`.
- Build metadata in `CMakeLists.txt`.

## Публичный API

Component-level collision API is installed through this package and participates in the canonical collision component namespace.
