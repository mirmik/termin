# termin-components-voxels

`termin-components-voxels` contains scene/render components for voxel grids:

- `termin.voxels.component.VoxelGridComponent`
- `termin.voxels.display_component.VoxelDisplayComponent`
- `termin.voxels.visualization.VoxelVisualizer`
- `termin.voxels.voxelizer_component.VoxelizerComponent`

The package depends on `termin-voxels` for voxel data and algorithms and on
scene/render/material packages for runtime component behavior.
