# termin-entity

`termin-entity` содержит Python-facing entity/component bindings для high-level пакета `termin.entity`.

Связанные документы:

- [Module Map](../../docs/modules.md#termin-entity)
- [termin-scene](../../termin-scene/docs/index.md)

## Основные области

- Python package в `python/termin/entity`.
- Packaging metadata в `setup.py` / `pyproject.toml`.

## Публичный API

Python package: `termin.entity` через пакет `termin-entity`.

Этот пакет является тонким Python distribution layer вокруг entity/component API, который используется остальными модулями и приложением.
