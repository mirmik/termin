# Foliage Test Assets

Small source assets for testing Termin foliage painting.

These files are project-style assets, not package assets. To try them in an
editor project, copy or import `Models/LowPolyBush.obj`,
`Models/LowPolyBush.obj.meta`, and `Materials/LowPolyBush.material` into that
project.

`LowPolyBush.obj` is a deliberately cheap opaque cross-card mesh:

- footprint: about 0.62 x 0.54 units;
- height: 0.60 units;
- origin: bottom center, with vertices starting at z=0.05;
- triangles: 60, double-sided through duplicated reversed faces.
