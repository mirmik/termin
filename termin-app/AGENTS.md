# AGENTS.md

Этот файл дополняет корневой `AGENTS.md` указаниями для `termin-app`.

## Направление развития

`termin-app` исторически был путанным монолитом. Мы постепенно выносим части системы во внешние по отношению к `termin-app` библиотеки, но процесс ещё не завершён.

В перспективе `termin-app` должен содержать только редактор-специфичный, окологуишный код.

Реэкспорты следует постепенно разбирать, перенаправляя импорты на канонические имена.
