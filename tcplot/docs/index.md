# tcplot

`tcplot` - lightweight plotting library поверх `tgfx2` и `tcgui`.

Связанные документы:

- [Module Map](../../docs/modules.md#tcplot)
- [termin-graphics](../../termin-graphics/docs/index.md)
- [termin-gui](../../termin-gui/docs/index.md)
- [tcplot C++ migration](../../termin-graphics/docs/migration-tcplot-cpp.md)

## Основные области

- Public C++ headers в `include/tcplot/`.
- Implementation в `src/`.
- Python bindings в `python/bindings/`.
- Python package в `python/tcplot/`.
- Examples в `examples/`.

## Публичный API

Python package:

```python
import tcplot
```

Examples are the current practical entrypoint for expected usage.
