# AGENTS.md

Этот файл дополняет корневой `AGENTS.md` указаниями для `termin-csg`.

## Назначение standalone CAD

`csg_cad.py` является тестовым и отладочным полигоном для мини-редактора
процедурного меша. Основная модель документа, операции редактирования,
evaluation и визуальные проекции должны жить в общем Python-слое
`termin.csg`, чтобы их можно было использовать как из standalone CAD, так и из
редактора приложения.

В `termin-app` этот мини-редактор встроен в tcgui extension
`ProceduralMeshEditorExtension` из
`termin/editor_tcgui/procedural_mesh_editor_extension.py`. Поэтому изменения в
`csg_cad.py` не должны превращаться в отдельную несовместимую реализацию
редактора: standalone приложение служит местом быстрой проверки поведения,
но каноническая логика должна оставаться общей.

## Практика разработки

При добавлении операций документа обновляй вместе:

- `procedural_document.py` для сериализуемой модели;
- `document_edit.py` для пользовательских команд редактирования;
- `document_eval.py` для построения CSG-результата;
- `document_tree_model.py` и visual/raycast слой, если меняется то, что видит
  пользователь или интерактивный выбор.

Ошибки редактирования и evaluation должны логироваться через `tcbase.log`.
